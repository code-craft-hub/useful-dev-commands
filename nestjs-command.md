## Nestjs useful commands

nest g app

nest g library (alias: lib)

nest g module (alias: mo)

nest g controller (alias: co)

nest g service (alias: s)

nest g provider (alias: pr)

nest g gateway (alias: ga)

nest g guard (alias: gu)

nest g interceptor (alias: in)

nest g middleware (alias: mi)

nest g filter (alias: f)

nest g pipe (alias: pi)

nest g resolver (alias: r)

nest g resource (alias: res)

nest g class (alias: cl)

nest g interface (alias: itf)

nest g decorator (alias: d)
(Note: This is a special generator that can create class/method/property decorators.)



nest g application (internal alias for app, but included for completeness)


nest start

nest start --watch

nest start --debug

nest start --prod

nest start --path <path>

nest start --entryFile <file>

nest start --config <tsconfig>
